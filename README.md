# ระบบบันทึกบิลและใบแจ้งหนี้หอพัก (Dormitory Billing System)

ระบบคำนวณและออกบิลเรียกเก็บเงินหอพักขนาด 75 ห้อง (5 ชั้น ชั้นละ 15 ห้อง: 101-115, 201-215, 301-315, 401-415, 501-515) พัฒนาด้วยเทคโนโลยี Vanilla HTML, CSS และ JavaScript มีระบบบันทึกค่าน้ำ ค่าไฟ ค่าส่วนกลาง และระบบเพิ่มรายการพิเศษเพิ่มเติม เช่น ค่าเช่าเครื่องใช้ไฟฟ้า ค่าซ่อมแซม หรือส่วนลดต่าง ๆ ได้ตามต้องการ

---

## ฟีเจอร์หลัก (Main Features)

1. **Dashboard สรุปภาพรวม**:
   - รายงานสถิติจำนวนห้องที่บันทึกบิลแล้ว, จ่ายเงินแล้ว, ค้างชำระ และคำนวณยอดเงินรวมทั้งหมดแบบเรียลไทม์
   - แถบความคืบหน้าแสดงสัดส่วนการชำระเงินของรอบบิลนั้นๆ

2. **ระบบจัดการค่าห้องพัก & ค่าน้ำค่าน้อย**:
   - แบ่งกริดห้องพักตามชั้น (ชั้น 1 ถึง ชั้น 5) เพื่อความรวดเร็วในการไล่จดมิเตอร์
   - ค้นหาเลขห้องและคัดกรองตามสถานะ (ยังไม่ทำบิล, ค้างชำระ, จ่ายเงินแล้ว) ได้
   - โหลดข้อมูลเลขมิเตอร์น้ำ/ไฟของรอบบิลก่อนหน้าขึ้นมาคำนวณโดยอัตโนมัติ

3. **ระบบเพิ่มรายการพิเศษแบบอิสระ (Dynamic Custom Charges)**:
   - มีปุ่ม `เพิ่มรายการ` เพื่อกรอกชื่อรายการและจำนวนเงินเพิ่มเติมสำหรับเฉพาะแต่ละห้องได้ไม่จำกัด (เช่น ค่าเช่าตู้เย็น, ค่าแอร์, ค่าปรับความเสียหาย)

4. **ใบแจ้งหนี้ / ใบเสร็จรับเงินพร้อมสั่งพิมพ์**:
   - แสดงตัวอย่างเอกสารก่อนพิมพ์
   - คำนวณราคาสุทธิและแปลงค่าเงินตัวเลขเป็นตัวอักษรภาษาไทยโดยอัตโนมัติ (เช่น *สามพันห้าร้อยบาทถ้วน*)
   - จัดรูปแบบสวยงามเพื่อรองรับการสั่งพิมพ์เป็นกระดาษ A4 หรือบันทึกเป็นไฟล์ PDF ผ่านคำสั่งพิมพ์ของเบราว์เซอร์ (`Ctrl + P`)

5. **ความปลอดภัยของข้อมูล & การพกพา**:
   - บันทึกข้อมูลอัตโนมัติลงในเบราว์เซอร์ผ่าน `localStorage` (ข้อมูลไม่หายเมื่อปิดเครื่องหรือรีเฟรชหน้าเว็บ)
   - ฟังก์ชัน **Export JSON** เพื่อดาวน์โหลดไฟล์สำรองข้อมูลลงเครื่องคอมพิวเตอร์ของคุณ
   - ฟังก์ชัน **Import JSON** เพื่อนำข้อมูลเก่ากลับมาใช้หรือย้ายเครื่องใช้งาน
